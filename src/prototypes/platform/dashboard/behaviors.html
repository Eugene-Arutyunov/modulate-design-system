Behaviors

Configure default API behaviors for your organization. These settings apply to all requests unless overridden per-request.

Transcription

Identify different speakers in the audio by default.

Detect synthetic or AI-generated voice per utterance.

Detect emotional tone (e.g. Calm, Happy, Frustrated) for each speaker turn.

Detect speaker accent (e.g. American, British).

Automatically tag and redact personally identifiable and health information.

Output

Include word-level timestamps in all transcription responses.

Return per-word confidence scores alongside transcription output.

Default output formats returned with each transcription request.

Moderation

Mask profanity in transcription output by default.

Append a content warning flag to responses containing sensitive material.
